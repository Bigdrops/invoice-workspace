What this is

BGD UI is the offline-first shell that houses every screen BIGDROPS ships. It's not a component library — it's a gallery of finished design directions, kept here so each one stays reviewable on its own terms.

How workspaces work

Every workspace owns its full identity: type pairing, palette, spacing, motion. Nothing is inherited from the shell beyond the grid that holds it. Invoicing shouldn't look like inventory, and here, it doesn't.
