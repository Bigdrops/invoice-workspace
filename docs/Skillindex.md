# Skill Index

All skills available via slash commands. Type `/` followed by the skill name to trigger.

---

## Android

| Skill | Description | When to Use |
|-------|-------------|-------------|
| `/android-architecture-clean` | Clean architecture boundaries, use cases, repositories, lifecycle-aware presentation models | Building or refactoring Android apps with clean architecture |
| `/android-coroutines-flow` | Coroutines, Flow, structured concurrency, dispatchers, cancellation-safe async pipelines | Working with Android async code, coroutines, or Flow |
| `/android-di-hilt` | Hilt dependency injection, scopes, testing overrides, module ownership boundaries | Wiring up Hilt DI in Android projects |
