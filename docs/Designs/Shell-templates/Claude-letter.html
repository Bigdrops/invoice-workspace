What this is

BGD UI is the personal, offline-first shell that houses every screen BIGDROPS ships. It's not a component library — it's a gallery of finished design directions, kept here so each one stays reviewable on its own terms before it's wired into the product.

How workspaces work

Every workspace owns its full identity: type pairing, palette, spacing, motion. Nothing here is inherited from the shell beyond the grid that holds it. That's deliberate — invoicing shouldn't look like inventory, and it doesn't.
